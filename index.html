なたね

Natane

Nataneは、日本語学習者コーパスに誤りを付与し、全文検索と誤り種別検索が可能なシステムです。

Natane is both a Japanese learner corpus annotated with errors and a system allowing full-text and error type search.

Nutmeg is a composition assistance system focused on technical writing that can check writing for non-technical language use.
